# Glossary

HTML: Structure of a webpage  
CSS: Styling of a webpage  
JavaScript: Behavior of a webpage  
DOM: Document Object Model  
Metadata: Information about the page  
Accessibility: Making sites usable for all  
Responsive Design: Adapting to screen size  
Form: User input structure  
Label: Text describing input  
Alt Text: Image description  
Viewport: Visible screen area
