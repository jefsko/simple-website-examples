    # Simple Website Series — Project 19: Project Documentation

    ## Overview
    This project is part of the **Simple Website Series**, a set of 19 small, incremental examples that introduce core website concepts step by step.

    Wraps the series with documentation that helps readers understand the projects, file structure, learning path, and terminology.

    ## What You'll Learn
    - Why documentation matters in real projects
    - How reference files support learning
    - How to navigate a multi-project series more confidently

    ## How to Run
    1. Open the documentation files in your editor or on GitHub
2. Start with `README.md`, then use the other markdown files as supporting references

No build tools or dependencies are required.

    ## Project Structure
    ```text
README.md
file-structure-reference.md
glossary.md
learning-path.md
project-summary.md
```
