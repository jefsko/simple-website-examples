# Simple Website Series

This repository is intended to act as the **main entry point** for the full Simple Website Series.

The series contains **19 small website projects** that build on one another in a clean, beginner-friendly order. Each project stays intentionally simple so it can be used as a reference, a teaching aid, or a starting point for experimentation.

## What This Series Is
- A sequential set of beginner-friendly website examples
- A plain HTML, CSS, and JavaScript learning path
- A minimal reference series that grows one concept at a time

## Who This Is For
This series is especially helpful for beginners, but it can also be useful for intermediate or advanced developers who want a clean set of stripped-down reference examples.

## How to Use This Repository
1. Start with Project 1 and move forward in order if you want the full learning path.
2. Jump to a specific project if you want a focused example of one topic.
3. Use the full reference document for the deeper cross-project summary and file listings.

## Learning Path Summary
- Projects 1–3: basic page structure and supporting assets
- Projects 4–5: navigation, multiple pages, and maintainability
- Projects 6–7: forms and validation
- Projects 8–10: images, responsiveness, and page metadata
- Projects 11–13: accessibility, usability, and richer metadata
- Projects 14–18: JavaScript organization, more advanced forms, refined accessibility, responsive refinement, and richer media
- Project 19: project-level documentation for the entire series

## Notes
- The projects are intentionally simple rather than production-ready.
- The goal is clarity, not completeness.
- Most projects can be opened directly in a browser without a build step.

## Suggested Companion Files
