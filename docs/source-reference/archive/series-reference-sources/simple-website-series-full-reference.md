# Simple Website Series — Full Project Reference (Refined)

## Overview
This document provides a complete reference for the **Simple Website Series**, a collection of 19 progressively built projects designed to demonstrate core web development concepts using plain HTML, CSS, and JavaScript.

The goal of this series is to offer a **clear, minimal, and practical learning path**:
- No frameworks
- No build tools
- No unnecessary complexity

Each project is intentionally simple and builds directly on the previous one.

---

## Who This Is For
This series is designed primarily for:
- Beginners learning web development from scratch
- Developers who want a clean reference without abstraction
- Anyone looking to revisit core fundamentals in a structured way

---

## How to Use This Series
You can approach this series in different ways:

- **Start from Project 1** if you're new — follow the progression step-by-step
- **Jump to specific topics** if you already know the basics
- **Use as a reference** alongside your own projects

Each project is self-contained, but together they form a complete learning path.

---

## Learning Progression
The projects are grouped into core phases:

1. **HTML Foundations** (Projects 1–5)
2. **Structure & Content** (Projects 6–10)
3. **Styling & Layout** (Projects 11–13)
4. **JavaScript & Interactivity** (Projects 14–17)
5. **Polish & Completion** (Projects 18–19)
