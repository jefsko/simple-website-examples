# Website Learning Series (Projects 1–18)

Start with Project 1 and move forward in order. Each project builds on the previous one and introduces one new concept at a time.

## What this is

A beginner-friendly series of small website projects designed to teach core web development concepts step by step.

## How to use this series

- Follow projects in order (recommended)
- Or jump to specific topics if needed
- Each project builds incrementally

## Project Overview

| Project | Name | Main Concept |
|---|---|---|
| 1 | First Web Page | Basic HTML structure |
| 2 | Basic Website | Metadata, robots, sitemap |
| 3 | Website Assets | CSS and JavaScript basics |
| 4 | Multi-Page Site | Navigation |
| 5 | Maintenance Basics | Organization |
| 6 | User Input | Basic forms |
| 7 | Advanced Forms | Validation |
| 8 | Images and Media | Images |
| 9 | Responsive Design | Layout adaptation |
| 10 | Head and Metadata | Metadata |
| 11 | Accessibility Basics | Labels, alt, links |
| 12 | Navigation and Usability | UX improvements |
| 13 | Advanced Metadata | Open Graph, robots |
| 14 | JavaScript Organization | Code structure |
| 15 | Expanded Forms | More inputs |
| 16 | Accessibility Refinements | Skip links, focus |
| 17 | Responsive Refinements | Breakpoints |
| 18 | Media and Embeds | Audio, video, iframe |

## What you will learn

- HTML structure
- CSS styling
- JavaScript basics
- Accessibility
- Responsive design
- Metadata
- Forms

## What this series does NOT cover

- Frameworks
- Backend development
- Build tools
- Advanced SEO

## What to do next

- Build your own website
- Explore frameworks
- Learn backend basics
