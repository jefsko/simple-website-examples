# Project Summary

This document summarizes the 18 website-code examples in the Simple Website Examples series.

| # | Project | Purpose |
|---:|---|---|
| 01 | [First Web Page](https://github.com/jefsko/simple-website-01-first-web-page) | Introduce basic HTML document structure and visible page content. |
| 02 | [Basic Site Files](https://github.com/jefsko/simple-website-02-basic-site-files) | Add foundational website files and basic metadata around a simple page. |
| 03 | [HTML, CSS, and JavaScript Assets](https://github.com/jefsko/simple-website-03-html-css-js-assets) | Introduce separate files for structure, presentation, and behavior. |
| 04 | [Multi-Page Site](https://github.com/jefsko/simple-website-04-multi-page-site) | Show how multiple pages, navigation, and shared styling fit together. |
| 05 | [Maintainable File Structure](https://github.com/jefsko/simple-website-05-maintainable-file-structure) | Improve organization by separating shared and page-specific styles. |
| 06 | [Basic Forms](https://github.com/jefsko/simple-website-06-basic-forms) | Introduce user input, labels, form submission handling, and simple feedback. |
| 07 | [Form Validation](https://github.com/jefsko/simple-website-07-form-validation) | Add basic client-side form validation and feedback patterns. |
| 08 | [Images and Gallery](https://github.com/jefsko/simple-website-08-images-and-gallery) | Introduce image assets, alt text, a gallery page, and image organization. |
| 09 | [Responsive Layout Basics](https://github.com/jefsko/simple-website-09-responsive-layout-basics) | Introduce responsive design through flexible layout and a small-screen breakpoint. |
| 10 | [Head Metadata and Favicon](https://github.com/jefsko/simple-website-10-head-metadata-and-favicon) | Explain common head metadata and favicon setup. |
| 11 | [Accessibility Basics](https://github.com/jefsko/simple-website-11-accessibility-basics) | Introduce accessibility fundamentals such as labels, alt text, and clear link wording. |
| 12 | [Navigation and Usability](https://github.com/jefsko/simple-website-12-navigation-and-usability) | Improve page navigation, consistency, and basic usability patterns. |
| 13 | [Open Graph and Metadata](https://github.com/jefsko/simple-website-13-open-graph-and-metadata) | Expand page metadata for sharing, author information, indexing hints, and previews. |
| 14 | [JavaScript Organization](https://github.com/jefsko/simple-website-14-javascript-organization) | Show more maintainable JavaScript structure with setup, handlers, and helpers. |
| 15 | [Expanded Form Controls](https://github.com/jefsko/simple-website-15-expanded-form-controls) | Introduce additional common form controls and clearer form layout. |
| 16 | [Accessibility Refinements](https://github.com/jefsko/simple-website-16-accessibility-refinements) | Refine accessibility with skip links, visible focus, labels, and image descriptions. |
| 17 | [Responsive Layout Refinements](https://github.com/jefsko/simple-website-17-responsive-layout-refinements) | Refine responsive behavior with multiple breakpoints and improved layouts. |
| 18 | [Media and Embeds](https://github.com/jefsko/simple-website-18-media-and-embeds) | Introduce audio, video, embedded content, and media file organization. |

## Important note about the previous Project 19 material

The original documentation folder that appeared as Project 19 did not contain website source code. It contained Markdown reference documents only. That content has been reorganized into this hub repository because the series currently contains 18 website examples and one hub documentation set.
